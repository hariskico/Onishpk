# ONI Website Demo (SQ/EN) — Static

This is a clean enterprise **static demo** (single-page) with a **one-click SQ/EN language toggle**.

## Run locally
Open `index.html` in a browser.

## Deploy on GitHub Pages
1. Create a GitHub repo (or use an existing one)
2. Upload the folder contents (index.html + assets/)
3. Go to **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: `main` (or your branch) and folder: `/ (root)`
6. Save

Your demo will appear at:
`https://<username>.github.io/<repo>/`

## Notes
- Logos are placeholders in this demo. Add files in `assets/` and replace the header block with `<img>` tags.
- Contact form is front-end only; wire it to your backend/CRM/email.
